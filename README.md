# embedding-bias-geometry

Geometric analysis of bias and semantic distortion in word embeddings.

## Goals
- SPIRE (Hood College): early framework + pilot evidence
- arxiv Publication
- AAAI 2027: geometry-first evaluation + bias–utility trade-off analysis

## Repo structure
- `src/` core code (load embeddings, debias transforms, metrics)
- `experiments/` runnable scripts + configs
- `research-log/` weekly research logbook
- `wordlists/` definitional pairs, neutral words, eval vocab lists
- `figures/` exported plots
- `paper/` draft sections / LaTeX or doc
