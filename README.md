\# embedding-bias-geometry



Geometric analysis of bias and semantic distortion in word embeddings.



\## Structure

\- `src/` core code (loading embeddings, debiasing transforms, metrics)

\- `experiments/` runnable scripts and configs

\- `research-log/` weekly research log

\- `wordlists/` definitional pairs, neutral words, eval vocab lists

\- `figures/` exported plots

\- `paper/` draft sections / latex or doc
